# 仿IOS风格弹窗插件
> 该插件基于 jquery 开发，UI界面仿造IOS的UIAlertView，相似度达到99%左右，可以达到以假乱真的效果，使用该弹窗可提升 WebApp 的视觉效果。

预览效果：https://funcodingofwe.github.io/jquery.UIAlertView/index.html
